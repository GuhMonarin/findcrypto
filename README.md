## 🚀 Como rodar localmente

### Pré-requisitos
- [Node.js](https://nodejs.org/) v18+
- [Docker](https://www.docker.com/)

### Passos
```bash
# Clone o repositório
git clone https://github.com/GuhMonarin/findcrypto

# Acesse a pasta do projeto
cd findcrypto

# Instale as dependências
npm install

# Inicie a aplicação
npm run dev
```
